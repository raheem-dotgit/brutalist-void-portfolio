# Brutalist Void Portfolio

A bold, brutalist-style portfolio template with smooth animations and 3D effects. Perfect for developers and designers who want to make a statement.

## Features

- Bold brutalist design aesthetic
- Custom cursor with hover effects
- Smooth scroll animations
- Section blend transitions
- Tech stack marquee
- Responsive layout
- Works showcase section
- Dark theme optimized

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **UI:** React 19, Tailwind CSS 4
- **3D Graphics:** Three.js, @react-three/fiber
- **Animations:** Framer Motion
- **Smooth Scroll:** Lenis
- **Language:** TypeScript

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm (recommended)

### Installation

```bash
# Clone the repository
git clone https://github.com/raheem-dotgit/brutalist-void-portfolio.git

# Navigate to the project directory
cd brutalist-void-portfolio

# Install dependencies
pnpm install

# Start the development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the project.

## Customization

1. Update your info in `components/about.tsx`
2. Add your projects in `components/works.tsx`
3. Modify the tech stack in `components/tech-marquee.tsx`
4. Update colors and fonts in `globals.css`
